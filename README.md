# 📰 The Dragon News

A modern, responsive **News Web Application** built with Next.js that delivers the latest news with authentication, category filtering, and detailed news views.

---

## 🚀 Live Demo

👉 https://dragon-news-eight-sand.vercel.app

---

## 🛠️ Technologies Used

### Frontend

- Next.js (App Router)
- React.js
- Tailwind CSS
- DaisyUI
- React Icons
- React Hook Form
- React Toastify

### Backend & Auth

- Better Auth (Authentication system)
- MongoDB (Database)
- Next.js API Routes

### Other Tools & Libraries

- date-fns (Date formatting)
- react-fast-marquee (Breaking news slider)

---

## ✨ Features of this Project

### 🔑 Authentication System

- User registration & login using email/password
- Social login with Google & GitHub
- Secure session handling
- Logout functionality

### 📰 News System

- Fetch news from external API
- Category-based filtering
- Dynamic routing for categories
- News details page with full content

### 🔐 Protected Routes

- Career page requires authentication
- News details page requires authentication
- Middleware-based route protection

### 📱 UI & UX

- Fully responsive design
- Modern UI with Tailwind & DaisyUI
- Loading spinner & error handling pages
- Custom 404 page

### 🧭 Navigation

- Dynamic navbar with active link highlight
- User profile display when logged in
- Conditional login/logout button

### 📊 Additional Pages

- About Page
- Career Page
- Authentication Pages (Sign In / Sign Up)
